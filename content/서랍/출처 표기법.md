# 내용
1. 책
	- 책 저자, 『책 제목』, 출판지-출판사(출판연도), 인용 쪽수.
	- 책 저자, 『책 제목』, 옮긴사람, 출판지-출판사(출판연도), 인용 쪽수.
	- 외국 책의 경우, 책 제목 작성 시 이탤릭체 사용
	- ex: 숀케 아렌스, 『제텔카스텐』, 김수진 옮김, 인간희극(2023), p.50
2. 글/논문
	- 논문 저자, "글/논문 제목”, 글/논문이 실린 간행물 제목, 발행호수 및 년도, 쪽수.
	- ex: 김성수, “한국 공연예술 행정의 당면과제에 대한 연구”. 중앙대학교 신문방송대학원 석사학위논문, 1990.
3. (인터넷) 신문 기사
	- 기자이름, “기사제목", 신문이름, 기사일자, (면수), (URL)
4. 웹페이지
	- 웹 페이지 이름, 날짜(최근 업데이트 날짜), URL
5. 사전 참고문헌
	- “항목”, 사전이름, 출판사(출판 연도)
